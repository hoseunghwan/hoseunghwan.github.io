# markdown grammar
***

#### 제목

<pre>
# 제목1
## 제목2
### 제목3
#### 제목4
##### 제목5
###### 제목6
</pre>

#### 인용

<pre>
> 인용
</pre>

#### 리스트

<pre>
* 리스트
- 리스트
+ 리스트
    * 리스트
    - 리스트
    + 리스트
</pre>

#### 번호리스트

<pre>
1. 리스트
2. 리스트
3. 리스트
</pre>

#### Italic

<pre>
*이탤릭*
_이탤릭_
</pre>

#### 제목

<pre>
**강조**
</pre>

#### Escape

<pre>
이스케이프 문자는 \를 붙여 표기합니다.
\\      backslash
\`      backtick
\*      asterisk
\_      underscore
\{\}    curly braces
\[\]    square brackets
\(\)    parentheses
\#      hash mark
\+      plus sign
\-       minus sign (hyphen)
\.       dot
\!       exclamation
</pre>

#### 코드

<pre>

</pre>

#### 코드블럭

<pre>
빈칸 4개로 코드블럭 시작
</pre>

#### 코드 하이라이트

<pre>
```html(해당언어)
<html>
<head>
    <title> 하하하 </title>
</head>
<body>
</body>
</html>
```
</pre>

#### 테이블

<pre>
| Left-Aligned  | Center Aligned  | Right Aligned |
| :------------ |:---------------:| -----:        |
| col 3 is      | some wordy text | $1600         |
| col 2 is      | centered        |   $12         |

</pre>
